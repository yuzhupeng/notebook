# IDA快捷键

P: 生成函数

C:生成汇编代码

U: 取消定义

D: 生成2/4/8字节的数据

A :形成字符串

N 重命名

H: 16进制转化

分号; 在汇编代码那里注释

shift + 分号 : 在汇编那里作为一个全局的注释

```c
 qmemcpy(v10, &unk_415A60, 0x17u);
  switch ( a2 )

    case 272:
```
